# تنفيذ ميزة وضع الضيف (Guest Mode) ✅

## ملخص التنفيذ

تم تنفيذ ميزة "وضع الضيف" بنجاح للتطبيق، والتي تسمح للعملاء المحتملين بتصفح التطبيق ومشاهدة بيانات تجريبية دون الحاجة لتسجيل الدخول.

---

## ✅ ما تم تنفيذه

### 1. إنشاء نظام وضع الضيف الكامل
**الملف:** `lib/interfaces/client/features/guest_mode/guest_mode.dart`

#### المكونات:
- **GuestModeProvider**: 
  - تفعيل/إلغاء تفعيل وضع الضيف
  - توفير بيانات تجريبية للعقود (3 عقود)
  - توفير بيانات تجريبية للتقارير (5 تقارير)
  - عرض حوار تسجيل الدخول

- **GuestModeInterceptor**:
  - اعتراض التفاعلات في وضع الضيف
  - عرض رسالة تطلب تسجيل الدخول عند المحاولة

### 2. إضافة زر "تصفح كضيف" في شاشة تسجيل الدخول
**الملف:** `lib/shared/features/auth/presentation/screens/login_view.dart`

#### التغييرات:
- زر جديد للعملاء فقط
- تصميم Outlined Button جذاب
- أيقونة + نص
- يفعل وضع الضيف ويوجه للشاشة الرئيسية

### 3. تعديل شاشة العميل لدعم وضع الضيف
**الملف:** `lib/interfaces/client/features/home/presentation/pages/customer_screen.dart`

#### التغييرات:
- فحص حالة وضع الضيف عند البناء
- عرض بيانات تجريبية في وضع الضيف
- عرض بيانات حقيقية للمستخدمين المسجلين
- بانر في أعلى الشاشة للتوضيح
- شاشة خاصة للملف الشخصي في وضع الضيف

### 4. حماية التفاعلات في العقود
**الملف:** `lib/interfaces/client/features/home/presentation/widgets/contract_card.dart`

#### التغييرات:
- إضافة فحص عند الضغط على البطاقة
- عرض حوار تسجيل الدخول في وضع الضيف

### 5. حماية التفاعلات في التجديد
**الملف:** `lib/interfaces/client/features/home/presentation/pages/renewal_screen.dart`

#### التغييرات:
- منع تجديد العقود في وضع الضيف
- عرض حوار تسجيل الدخول عند المحاولة

## 📊 البيانات التجريبية

### العقود (3 عقود):
1. **CNT-2024-001** - عقد نشط (مصر الجديدة)
2. **CNT-2024-002** - عقد نشط (التجمع الخامس)
3. **CNT-2023-015** - عقد منتهي (المهندسين)

### التقارير (5 تقارير):
1. صيانة دورية للعقد الأول
2. صيانة طارئة للعقد الأول
3. صيانة دورية للعقد الثاني
4. فحص دوري للعقد الثاني
5. صيانة نهائية للعقد الثالث

### معلومات إضافية:
- **العميل التجريبي**: شركة النموذج التجريبي
- **الفنيون**: أحمد محمد، محمد علي، خالد حسن
- **التواريخ**: تواريخ نسبية (حديثة ومنطقية)

## 🔒 الأمان والحماية

### ما يُسمح به في وضع الضيف:
- ✅ عرض العقود
- ✅ عرض التقارير
- ✅ التنقل بين الصفحات
- ✅ قراءة البيانات

### ما لا يُسمح به:
- ❌ الضغط على العقود

- ❌ عرض الملف الشخصي الحقيقي
- ❌ أي عمليات كتابة أو تعديل

## 🧪 الاختبار

### خطوات الاختبار:
1. شغل التطبيق
2. اختر "عميل"
3. اضغط "تصفح كضيف"
4. ✅ يجب أن تشاهد شاشة العميل مع بانر
5. ✅ يجب أن تشاهد 3 عقود
6. اضغط على تبويب "التقارير"
7. ✅ يجب أن تشاهد 5 تقارير
8. حاول الضغط على أي عقد
9. ✅ يجب أن يظهر حوار تسجيل الدخول
10. اذهب لتبويب "تجديد"
11. حاول تجديد عقد
12. ✅ يجب أن يظهر حوار تسجيل الدخول

## 🚀 طريقة الاستخدام في الكود

### تفعيل وضع الضيف:
```dart
GuestModeProvider.enableGuestMode();
context.go(Routes.customerScreen);
```

### التحقق من الحالة:
```dart
if (GuestModeProvider.isGuestMode) {
  // في وضع الضيف
}
```

### حماية إجراء:
```dart
onTap: () {
  if (!GuestModeInterceptor.checkAndShowLoginDialog(context)) {
    return;
  }
  // تنفيذ الإجراء
}
```

### البيانات التجريبية:
```dart
final contracts = GuestModeProvider.getDummyContracts();
final reports = GuestModeProvider.getDummyReports();
```
