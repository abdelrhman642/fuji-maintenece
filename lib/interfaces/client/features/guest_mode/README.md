# Guest Mode Feature - وضع الضيف

## الوصف
ميزة "وضع الضيف" تتيح للعملاء المحتملين تصفح التطبيق ومشاهدة بيانات تجريبية دون الحاجة لتسجيل الدخول.

## المميزات

### 1. الدخول كضيف
- يظهر زر "تصفح كضيف" في شاشة تسجيل الدخول للعملاء فقط
- يتم تفعيل وضع الضيف بالضغط على الزر
- يتم توجيه المستخدم مباشرة إلى شاشة العميل

### 2. البيانات التجريبية
تتضمن البيانات التجريبية:

#### العقود (Contracts)
- 3 عقود نموذجية:
  - عقدين نشطين (active)
  - عقد منتهي (expired)
- تفاصيل كاملة لكل عقد (رقم العقد، التواريخ، الموقع، إلخ)

#### التقارير (Reports)
- 5 تقارير نموذجية:
  - تقارير صيانة دورية
  - تقارير صيانة طارئة
  - تقارير فحص دوري
  - تقرير صيانة نهائية
- معلومات الفنيين والأسئلة والأجوبة

### 3. حماية التفاعل
- عند محاولة الضغط على أي عنصر أو التفاعل معه:
  - يظهر حوار يطلب من المستخدم تسجيل الدخول
  - خيارات: إلغاء أو تسجيل الدخول
  - إذا اختار تسجيل الدخول، يتم توجيهه لشاشة اللوجن

### 4. واجهة المستخدم
- بانر في أعلى الشاشة يوضح أن المستخدم في وضع الضيف
- زر "تسجيل الدخول" في البانر للوصول السريع
- شاشة خاصة للملف الشخصي في وضع الضيف

## الملفات المضافة/المعدلة

### 1. الملف الرئيسي
```
lib/interfaces/client/features/guest_mode/guest_mode.dart
```
يحتوي على:
- `GuestModeProvider`: يوفر البيانات التجريبية والحالة
- `GuestModeInterceptor`: يعترض التفاعلات ويطلب تسجيل الدخول

### 2. الملفات المعدلة
- `lib/shared/features/auth/presentation/screens/login_view.dart`
  - إضافة زر "تصفح كضيف" للعملاء

- `lib/interfaces/client/features/home/presentation/pages/customer_screen.dart`
  - دعم وضع الضيف
  - عرض البيانات التجريبية
  - إضافة بانر وضع الضيف

- `lib/interfaces/client/features/home/presentation/widgets/contract_card.dart`
  - إضافة interceptor للتحقق من وضع الضيف

- `lib/interfaces/client/features/home/presentation/pages/renewal_screen.dart`
  - إضافة interceptor لمنع تجديد العقود في وضع الضيف

## طريقة الاستخدام

### تفعيل وضع الضيف
```dart
GuestModeProvider.enableGuestMode();
```

### إلغاء تفعيل وضع الضيف
```dart
GuestModeProvider.disableGuestMode();
```

### التحقق من وضع الضيف
```dart
if (GuestModeProvider.isGuestMode) {
  // المستخدم في وضع الضيف
}
```

### حماية إجراء معين
```dart
onTap: () {
  if (!GuestModeInterceptor.checkAndShowLoginDialog(context)) {
    return; // توقف التنفيذ إذا كان في وضع الضيف
  }
  // تنفيذ الإجراء للمستخدمين المسجلين
}
```

### الحصول على البيانات التجريبية
```dart
// الحصول على العقود التجريبية
final contracts = GuestModeProvider.getDummyContracts();

// الحصول على التقارير التجريبية
final reports = GuestModeProvider.getDummyReports();
```

## ملاحظات تطويرية

### البيانات التجريبية
- يتم إنشاء البيانات التجريبية برمجياً باستخدام تواريخ نسبية
- العميل التجريبي: "شركة النموذج التجريبي"
- الفنيين التجريبيين: أحمد محمد، محمد علي، خالد حسن

### التوافق
- البيانات التجريبية متوافقة مع نماذج البيانات الحقيقية
- تحويل `ContractModel` إلى `Contract` للتقارير

### الأمان
- لا يتم حفظ أي بيانات في وضع الضيف
- يتم منع جميع عمليات الكتابة والتعديل
- لا يمكن الوصول للملف الشخصي الحقيقي

## المستقبل
- إضافة المزيد من البيانات التجريبية
- إضافة جولة تعريفية (Tour) للمستخدمين الجدد
- تتبع سلوك المستخدم في وضع الضيف (Analytics)
- إضافة خيار "تحويل إلى حساب حقيقي"

## الاختبار
لاختبار الميزة:
1. افتح التطبيق
2. اختر "عميل" في شاشة اختيار نوع الحساب
3. اضغط على زر "تصفح كضيف"
4. تصفح البيانات التجريبية
5. حاول الضغط على أي عنصر - يجب أن يظهر حوار تسجيل الدخول

## الدعم
للأسئلة والمشاكل، يرجى التواصل مع فريق التطوير.
